CTF Challenges for Lab 2 of ML for Cybersecurity.  
Please create a single Agentic Framework which utilizes an LLM backbone to retrieve the flag in each challenge.  
We recommend you include pwntools and the radare2 decompiler plugin, r2dec,  in the loop of your framework.  
Examples of agentic frameworks can be found in the [agentic_demos](https://github.com/ajn313/ML-CyberSec-2025-Lab2-Public/tree/main/agentic_demos) folder.
